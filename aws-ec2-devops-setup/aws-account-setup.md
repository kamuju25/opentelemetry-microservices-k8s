# AWS Account Creation

## Introduction

Setting up an AWS account is your first step into cloud computing. Below is a step-by-step guide to help you create your AWS account and begin using it.

## Step-by-Step AWS Account Setup

**Step 1: Open the AWS Registration Page**

  - Navigate to the AWS sign-up website [AWS-Sign-up](https://aws.amazon.com/) 
  - Click on Create an AWS Account  

`Step 2: Provide Email and Account Information`  

  - Enter a valid email address  
  - Choose a root username (your full name)  
  - Create a strong password  
  - Click Continue  

`Step 3: Enter Contact Details`

  - Select either Personal or Business account type
  - Fill in your name, phone number, and address
  - Click Continue

`Step 4: Add Payment Method`

  - Provide a valid credit or debit card (used for identity verification)
  - AWS may charge a small temporary amount for verification purposes
  - Click Continue

`Step 5: Verify Your Identity`

  - Enter the OTP received via SMS or voice call
  - Complete the CAPTCHA challenge
  - Click Verify

`Step 6: Select a Support Plan`

  - Choose the Basic (Free) support plan unless advanced support is needed
  - Click Continue

`Step 7: Log in to AWS Console`

  - After verification, access the AWS Management Console [AWS-Console](https://console.aws.amazon.com/)
  - Sign in using your registered email and password
  - Start exploring AWS services!
